# Angel

A modern theme for the [Ghost](https://ghost.org/) blogging platform.

## Overview

Angel is a clean, minimalist theme designed for Ghost blogs. It aims to provide a beautiful reading experience while maintaining excellent performance.

## Features

- Clean, modern design
- Responsive layout
- Fast loading times
- SEO optimized

## Installation

To install this theme on your Ghost blog:

```bash
# Clone this repository
git clone https://github.com/zzh8829/angel.git

# Copy the theme to your Ghost themes directory
cp -R angel /path/to/your/ghost/content/themes/
```

Then activate the theme in your Ghost admin panel under Settings > Design.

## Development

# Clone the repository
